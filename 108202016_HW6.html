2.What is the relation between band structure and DOS in your results?

The DOS is the density of the lines of the band structure for a specific energy. Integrating DOS to Fermi level will be equal to total number of occupied. The number of orbitals in a band within the first Brillouin zone is equal to the number of unit cells N in the crystal. This is much the same as the statement made in connection with the number of lattice vibrational modes, and is proved in a like manner, by appealing to the boundary conditions. Consider first the one-dimensional case. The allowed values of k form a uniform mesh whose unit spacing is 2/L. The number of states inside the first zone, whose length is 2/a, is therefore equal to (2/a)/(2/L) = L/a = N, where N is the number of unit cells, in agreement with the assertion made earlier.
